# PDF to Markdown Converter

![PDF to Markdown](https://raw.githubusercontent.com/U-C4N/pdftomarkdown/main/generated-icon.png)

A web-based PDF to Markdown converter that provides users with a professional document conversion experience. Convert your PDF files to Markdown format quickly and easily.

[🔗 Live Demo](https://pdftomarkdown.u-c4n.repl.co)

## ✨ Features

- 📄 Drag-and-drop file upload
- ⚡ Fast conversion process
- 📊 Smart token calculation
- 🎯 Precise header and list detection
- 📋 One-click copy functionality
- 🔄 Automatic table format conversion
- 📱 Mobile-responsive interface

## 🛠️ Technologies

- Python Flask
- PyPDF2
- Rich (Terminal UI)
- Modern HTML/CSS/JavaScript
- Bootstrap 5

## 🚀 Installation

1. Clone the repository:
```bash
git clone https://github.com/U-C4N/pdftomarkdown.git
cd pdftomarkdown
```

2. Install required packages:
```bash
pip install -r requirements.txt
```

3. Start the application:
```bash
python app.py
```

4. Open in your browser:
```
http://localhost:3000
```

## 💡 Usage

1. Open the application in your browser
2. Drag and drop or select your PDF file
3. Click the "Convert" button
4. Copy the generated Markdown content
